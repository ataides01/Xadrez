Xadrez em Python

Um jogo de xadrez completo desenvolvido em Python, com interface gráfica interativa, validação de jogadas oficiais e estrutura pronta para evolução (IA, efeitos visuais e melhorias de gameplay).

🚀 Tecnologias utilizadas
Python
pygame (interface gráfica)
python-chess (regras do jogo)
🎮 Funcionalidades

✔ Tabuleiro de xadrez 8x8
✔ Movimentação de peças via clique
✔ Validação de jogadas oficiais
✔ Controle automático de turnos
✔ Renderização gráfica das peças
✔ Estrutura modular para expansão

🖼️ Interface

O jogo apresenta um tabuleiro visual com peças estilizadas, permitindo interação direta com o mouse para movimentação.
